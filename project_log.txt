GadgetGrid - Project Execution Log
===================================
এই ফাইলে প্রতিটি কমপ্লিট হওয়া স্টেপের বিস্তারিত কাজ এবং টেকনোলজির বিবরণ লেখা হবে।

Step 1: Initialize Next.js project with TypeScript & Initialize Git Repository
-----------------------------------------------------------------------------
- Technologies Used: Next.js 15 (App Router), TypeScript, Git.
- How it was done: 
  * `create-next-app` কমান্ড ব্যবহার করে প্রজেক্ট তৈরি করা হয়েছে।
  * TypeScript, Tailwind CSS, ESLint, এবং App Router এনাবল করা হয়েছে।
  * গিট ইনিশিয়ালাইজ করে আপনার দেওয়া গিটহাব রিপোজিটরির (`sanoyon211/GadgetGrid`) সাথে লোকাল প্রজেক্ট কানেক্ট করা হয়েছে।
  * প্রথম কমিট করে `main` ব্রাঞ্চে পুশ করা হয়েছে।

Step 2: Configure Tailwind CSS, custom fonts (Outfit & Inter), and base CSS
-----------------------------------------------------------------------------
- Technologies Used: Tailwind CSS v4, Google Fonts (next/font/google).
- How it was done:
  * `src/app/layout.tsx` ফাইলটি মডিফাই করে `Outfit` (হেডিংয়ের জন্য) এবং `Inter` (সাধারণ লেখার জন্য) ফন্ট ইমপোর্ট করে যুক্ত করা হয়েছে।
  * `src/app/globals.css` ফাইলে Tailwind v4 এর `@theme` রুল ব্যবহার করে "Sunset Minimalist" থিমের কালার কোডগুলো গ্লোবাল ভ্যারিয়েবল হিসেবে অ্যাড করা হয়েছে (যাতে পুরো প্রজেক্টে সহজেই ব্যবহার করা যায়)।
  * কোডগুলো গিটহাবে পুশ করা হয়েছে।

Step 3 & 4: Setup UI dependencies & Design System
-----------------------------------------------------------------------------
- Technologies Used: npm, lucide-react, sonner, recharts.
- How it was done:
  * আইকন দেখানোর জন্য `lucide-react` ইনস্টল করা হয়েছে।
  * সুন্দর টোস্ট মেসেজের জন্য `sonner` ইনস্টল করা হয়েছে।
  * ড্যাশবোর্ডে চার্ট/গ্রাফ দেখানোর জন্য `recharts` ইনস্টল করা হয়েছে।
  * (নোট: Step 4 এর ডিজাইন সিস্টেম কনফিগারেশনের কাজ আমরা আগেই Step 2-এ Tailwind v4 এর মাধ্যমে `globals.css`-এ কমপ্লিট করে ফেলেছি)।
  * গিটহাবে পুশ করা হয়েছে।

Step 5: Build Global Components (Navbar & Footer)
-----------------------------------------------------------------------------
- Technologies Used: Next.js (App Router), Tailwind CSS, lucide-react.
- How it was done:
  * `src/components/globals/Navbar.tsx` তৈরি করে রেসপন্সিভ (মোবাইল হ্যামবার্গার মেনু) এবং স্টিকি নেভিগেশন বার ডিজাইন করা হয়েছে।
  * `src/components/globals/Footer.tsx` তৈরি করে ৪-কলাম বিশিষ্ট একটি সুন্দর ফুটার ডিজাইন করা হয়েছে।
  * `src/app/layout.tsx` ফাইলে Navbar এবং Footer ইমপোর্ট করে যুক্ত করা হয়েছে, যাতে এগুলো ওয়েবসাইটের প্রতিটি পেজে দেখায়।
  * গিটহাবে পুশ করা হয়েছে।

Step 6: Build Homepage Sections with Separate Components
-----------------------------------------------------------------------------
- Technologies Used: Next.js, Tailwind CSS, lucide-react.
- How it was done:
  * `src/components/home/HeroSection.tsx` তৈরি করে একটি আকর্ষণীয় হিরো ব্যানার ডিজাইন করা হয়েছে।
  * `src/components/home/FeaturedCategories.tsx` তৈরি করে বিভিন্ন গ্যাজেট ক্যাটাগরির গ্রিড তৈরি করা হয়েছে।
  * `src/components/home/TrendingGadgets.tsx` তৈরি করে ট্রেন্ডিং প্রোডাক্টের লিস্ট ডিজাইন করা হয়েছে।
  * `src/components/home/NewsletterSection.tsx` তৈরি করে ইমেইল সাবস্ক্রিপশনের অংশ তৈরি করা হয়েছে।
  * `src/app/page.tsx` থেকে ডিফল্ট Next.js কোড সরিয়ে এই সবগুলো কম্পোনেন্ট একসাথে যুক্ত করা হয়েছে।

Step 7: Build Explore/Shop Page
-----------------------------------------------------------------------------
- Technologies Used: Next.js (App Router), Tailwind CSS, React State.
- How it was done:
  * `src/components/shop/ProductFilters.tsx` তৈরি করে ক্যাটাগরি, দাম এবং রেটিং ফিল্টার করার জন্য একটি সাইডবার ফিল্টার মেনু ডিজাইন করা হয়েছে।
  * `src/components/shop/SearchAndSort.tsx` তৈরি করে একটি সার্চবার এবং প্রোডাক্ট সর্ট (Sort) করার ড্রপডাউন মেনু যুক্ত করা হয়েছে।
  * `src/components/shop/ProductGrid.tsx` তৈরি করে ডামি প্রোডাক্ট দিয়ে একটি সুন্দর ও রেস্পন্সিভ প্রোডাক্ট গ্রিড বানানো হয়েছে।
  * `src/app/products/page.tsx` এ একটি নতুন রাউট তৈরি করে উপরের সবগুলো কম্পোনেন্ট লেফট সাইডবার লেআউটে সাজানো হয়েছে।
